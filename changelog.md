# Changelog - consoleLog

## [2026.01.03]
### Cambios Principales
* **Rebranding Completo**: El Complemento ha sido renombrado de "VisorConsolasXD" a **consoleLog**, recuperando su identidad original con tecnología moderna.
* **Nueva Arquitectura Modular**: Se ha reescrito el núcleo del complemento para soportar un sistema de plugins extensible y una gestión de memoria más eficiente.
* **Sistema de Plugins (11 herramientas)**:
    * **Google AI (Gemini)**: Integración avanzada con modelos de IA para analizar y depurar contenido de consola. Soporte para múltiples API Keys y archivos adjuntos.
    * **Extractor de Datos**: Localización rápida de URLs, direcciones IP y rutas de archivos.
    * **JSON Beauty**: Formateador inteligente para bloques JSON detectados.
    * **Filtro de Log**: Aislamiento de líneas específicas para facilitar la depuración.
    * **Calculadora Express**, **Base64**, **Timestamp Converter** y más utilidades integradas.
* **Lanzador de Consolas Mejorado**:
    * Soporte completo para **CMD**, **PowerShell**, **Windows Terminal**, **Git Bash** y **Visual Studio Developer Command Prompt**.
    * **Memoria de selección**: Ahora el lanzador recuerda opcionalmente la última consola utilizada (configurable en opciones).
    * Posibilidad de abrir consolas como administrador directamente desde el diálogo.
* **Interfaz de Usuario Pro**:
    * Ventana de visor optimizada con barra de menús accesible nativa.
    * Mejor manejo de la tecla **Alt** para acceder a los menús sin interferencias.
    * Barra de estado con información en tiempo real de posición y estadísticas de texto.

### Correcciones y Mejoras
* Se corrigió un error que impedía que el lanzador recordara la última consola seleccionada.
* Optimización en la captura de texto para Windows Terminal.
* Actualización de la documentación completa (Markdown y HTML) con todos los atajos y nuevas funciones.
* Mejora en la compatibilidad con versiones recientes de NVDA (2025.1+).
